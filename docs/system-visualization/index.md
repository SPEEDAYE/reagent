# System Visualization

This directory maps the runnable system from code entry to outputs.

| File | Purpose |
|---|---|
| `code-functions.md` | Pure-code functions and core responsibilities. |
| `pipeline.md` | Ordered execution flow from `main.py` or API start to `experiment/<run-id>/`. |
| `llm-calls/` | CrewAI/LLM call records. |
| `prompt-sources.md` | Prompt and task configuration sources. |
| `external-calls.md` | API, database, filesystem, and network calls. |
